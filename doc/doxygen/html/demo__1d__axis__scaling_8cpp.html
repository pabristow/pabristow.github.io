Scale axis and update min and max axis values, and tick increment and number of ticks.

Parameters
min_valueScale axis from explicit input range minimum.
max_valueScale axis from explicit input range maximum.
axis_min_valueComputed minimum value for the axis, updated by scale_axis.
axis_max_valueComputed maximum value for the axis, updated by scale_axis.
axis_tick_incrementComputed tick increment for the axis, updated by scale_axis.
auto_ticksComputed number of ticks, updated by scale_axis.
originIf false, do not include the origin unless the range min_value to max_value includes zero.
tightFraction of overrun allowed before another tick used. For a good visual effect, up to about 0.001 might suit a 1000 pixel wide image, allowing values just 1 pixel over the tick to be shown.
min_ticksMinimum number of major ticks.
steps0, or 2 for 2, 4, 6, 8, 10, 5 for 1, 5, 10, or 10 (2, 5, 10).
